# My-Projects
Interactive Power BI weather dashboard using live API data for global conditions, trends, and real-time insights.

This Power BI dashboard delivers real-time global weather insights by connecting directly to a live weather API. It integrates actual temperature, humidity, wind, and condition data into a clean, visually compelling report designed for deeper trend analysis and dynamic comparisons across cities. The project demonstrates advanced data modeling, automated refresh logic, and optimized DAX measures to support accurate, time-sensitive reporting. It highlights practical skills in API integration, Power BI visualization design, and real-world analytics development.
